# Arosync Media — Short-Form Video Editor Portfolio

Single-page portfolio skeleton for showcasing 9:16 Reels / Shorts work.

## Stack

- HTML5 semantic structure
- Tailwind CSS (CDN) + light custom CSS
- Vanilla JavaScript (nav, video fallbacks)

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Customize

1. Replace brand name, email, and social URLs in `index.html`
2. Add vertical MP4s under `assets/videos/` (see that folder’s README)
3. Tweak colors in the Tailwind config block in `index.html` (`ink`, `accent`, `canvas`)
